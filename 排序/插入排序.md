# 插入排序

#### 【思想】

每步将一个待排序的对象，按其关键字的大小，插入到前面已经排好序的一组对象的适当位置，原来对象位置上的所有对象依次向后顺移，直到对象全部插完为止。

#### 【代码】

```c++
void insertSort(int r[], int n){
	for(i=2;i<=n;i++){
		r[0]=r[i];j=i-1;
		while(r[0]<r[j]){
			r[j+1]=r[j];
			j=j-1;
		}
		r[j+1]=r[0]
	}
} 
```

#### 【算法性能】

最好情况:O(n)

最坏情况：O($n^2$)

平均情况：O($n^2$)

空间性能：需要一个记录的辅助空间

直接排序算法是一种稳定的排序算法。
